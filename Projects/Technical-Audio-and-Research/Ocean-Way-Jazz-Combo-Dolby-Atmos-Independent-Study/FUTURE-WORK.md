# Future Work

## Compare immersive arrays

A future study could record the same ensemble using multiple immersive techniques, such as:

- 2L Cube
- OCT-3D
- PCMA-3D
- Decca Cuboid
- Hamasaki-based height arrays
- Ambisonic arrays

Using the same ensemble, repertoire, room, and signal path would allow more controlled comparison.

## Conduct listening tests

Listeners could compare the arrays using:

- A calibrated 7.1.4 system
- Binaural headphone renders
- Stereo translations

Testing could examine spatial clarity, envelopment, localization, tonal balance, listener preference, and translation between formats.

## Examine channel and object workflows

A follow-up project could compare:

- Fixed channel-based capture
- Object-based instrument placement
- Hybrid bed-and-object workflows

This would help determine when object automation contributes meaningfully and when the captured acoustic image is sufficient.

## Improve documentation

Future sessions should preserve:

- Complete microphone and input lists
- Array measurements and diagrams
- Performer locations
- Routing and I/O documentation
- Plug-in and renderer settings
- Session information exports
- Raw and final audio
- Binaural, stereo, and multichannel renders

## Reopen and analyze the original sessions

If compatible Pro Tools access becomes available, the original sessions could be reviewed to recover:

- Exact track counts
- Plug-in assignments
- Object and bed routing
- Pan settings
- Automation
- Renderer configuration
- Final deliverables

## Expected value

These extensions would help evaluate how array geometry, venue acoustics, channel routing, and playback format affect the perceived quality of immersive music recordings.

# MATLAB Projects

This folder contains MATLAB audio-software and digital signal processing work.

## Projects

- [MATLAB Audio DSP Fundamentals](MATLAB-Audio-DSP-Fundamentals/README.md) — pink-noise synthesis, FIR band-pass and band-stop filters, frequency-domain IIR filtering, and discrete-time system analysis
- [MATLAB Impulse-Response and Audio Systems Analysis](MATLAB-Impulse-Response-and-Audio-Systems-Analysis/README.md) — test-signal generation, convolution, FFT analysis, room-response estimation, and analog/digital system comparison
- [MATLAB Distortion and Audio Analysis](MATLAB-Distortion-and-Audio-Analysis/README.md) — nonlinear waveshaping, clipping, bit-depth reduction, THD, and supporting stereo-analysis exercises
- [MATLAB Convolution and Tempo-Synchronized Delay](MATLAB-Convolution-and-Tempo-Synchronized-Delay/README.md) — L-C-R ping-pong echo, sample-by-sample convolution, and FFT-domain convolution coursework
- [MATLAB Nonlinear Distortion and Mid-Side Processing](MATLAB-Nonlinear-Distortion-and-Mid-Side-Processing/README.md) — hard clipping, transistor-inspired saturation, parallel harmonic processing, and mid-side distortion

These projects preserve the original coursework and code while documenting technical limitations and stronger future implementations.

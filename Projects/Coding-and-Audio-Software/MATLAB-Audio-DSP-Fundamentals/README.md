# MATLAB Project — Audio DSP Fundamentals

## Project type

MATLAB audio-software and digital signal processing project completed by Kevin Worley for AET 5420 Audio Signal Processing at Belmont University.

## Objective

Implement foundational DSP operations in MATLAB through three related exercises:

1. Synthesize pink noise from generated white noise.
2. Design reusable FIR band-pass and band-stop filter functions.
3. Analyze a discrete-time FIR system using its difference equation, transfer function, frequency response, and zeros.

## Project contents

### MATLAB source

- [`prob2.m`](src/prob2.m) — white-noise generation, octave-based FIR shaping, convolution, plotting, and WAV export
- [`bandPass.m`](src/bandPass.m) — FIR band-pass processing using cascaded low-pass and high-pass stages
- [`bandStop.m`](src/bandStop.m) — FIR band-stop processing using parallel low-pass and high-pass stages
- [`problem2.m`](src/problem2.m) — impulse-based test script for the band-pass and band-stop functions
- [`bandTest.m`](src/bandTest.m) — audio-file test harness for the reusable filter functions
- [`problem3HW.m`](src/problem3HW.m) — preserved incomplete experimental script for the difference-equation problem

### Audio output

## DSP concepts demonstrated

- Random-signal generation
- Sample-rate and Nyquist-frequency calculations
- Frequency normalization
- FIR filter design with `fir1` and `fir2`
- Serial and parallel filter structures
- Convolution
- Impulse-response testing
- Log-frequency magnitude plotting
- Difference equations
- Z-transforms and transfer functions
- Pole-zero reasoning
- Modular MATLAB functions and test scripts

# Immersive Recording Workflow

## Capture before manipulation

The project treated immersive imaging primarily as a recording problem rather than a post-production effect.

The intended soundstage was created through:

1. Selecting the historic Ocean Way Nashville Studio A.
2. Positioning the ensemble relative to the main array.
3. Capturing front, side, rear, and height information.
4. Routing the captured positions to corresponding loudspeakers.
5. Using spot microphones to support instrumental clarity and balance.
6. Managing the LFE path separately within the 7.1.4 production.

## Channel-based presentation

The 2L Cube-inspired method used predictable channel-based placement. A microphone capturing a particular spatial region was associated with its corresponding playback location.

This approach preserved:

- Directional relationships
- Differences in arrival time
- Relative level relationships
- Room ambience
- Height information
- A stable listener perspective

## Use of objects and automation

The production did not depend on sounds moving around the listener. Object-routing elements existed in the sessions, but extensive pan and object automation was not necessary to achieve the project’s principal goal.

The desired immersive presentation came from the recorded geometry and stable routing rather than artificial motion.

## Bed structure

The surviving sessions expose eleven full-range spatial outputs corresponding to seven horizontal and four height positions.

The LFE path completing the 7.1.4 format was handled separately because LFE is not ordinarily captured as an independent position in the main microphone array.

## Translation

The project was designed for Dolby Atmos reproduction while following an array philosophy based on direct spatial correspondence. Future analysis could examine how successfully the same recording translates to binaural and stereo formats.
